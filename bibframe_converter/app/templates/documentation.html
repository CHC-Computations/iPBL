CSV Documentation

The input CSV file should have the following format:

subject,type,predicate,object
http://example.org/book1,http://schema.org/Book,http://schema.org/name,Example Book
http://example.org/book1,http://schema.org/Book,http://schema.org/author,John Doe
http://example.org/book1,http://schema.org/Book,http://schema.org/datePublished,2020-01-01

Each row in the CSV file represents a triple (subject, predicate, object) for the RDF graph.
